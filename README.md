# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Av0905/CovalentX-Technologies-Internship.git
```

Move inside project

```bash
cd CovalentX-Technologies-Internship
```

Open any task folder and run the project.

For the React Portfolio (Task 5):

```bash
npm install
```

```bash
npm run dev
```
